Directory Fuzzing

 

Learn how to discover new web pages and expand our attack surface by fuzzing for directories.

 

Key concepts 

- Enumeration

- Scanning

- Information Gathering

- Web App testing

- Fuzzing

 

Web directories

Web directories are like a local filesystem but for a website. We can use fuzzing techniques similar to the subdomain and vhost fuzzing module to change the page we are requesting.



We add to the start line and tell the server what page we are looking for. This becomes our payload and is what we change for each request we send.

 

Why is discovering web pages important?

Web discovery is important because it means we can find new pages in an application thus increasing our attack surface. New pages can contain sensitive information, or new functionality, both of which are very important to discover. Making sure we have a comprehensive list of all the pages we can find, means that we won’t miss any information or functionality when it comes to testing.
